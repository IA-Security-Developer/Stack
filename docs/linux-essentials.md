# Linux Essentials

## Objetivo

Aprender los comandos fundamentales de Linux Debian.

## Navegación

```bash
pwd
ls
ls -la
cd
tree
```

## Archivos

```bash
touch
nano
cat
mkdir
mv
cp
rm
```

## Permisos

```bash
chmod
chown
```

## Red

```bash
ss -tuln
ip a
ping
```

## Servicios

```bash
systemctl status ssh
```
